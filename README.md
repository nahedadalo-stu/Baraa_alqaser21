## اسم المشروع

نظام إدارة مؤسسة القصر التعليمية

---

# وصف المشروع

هو تطبيق ويب ديناميكي تم تطويره باستخدام لغة PHP وقاعدة بيانات MySQL بهدف إدارة الدورات التدريبية داخل مؤسسة تعليمية. يتيح النظام للمستخدمين إنشاء حسابات، تسجيل الدخول، استعراض الدورات، التسجيل فيها، ومتابعة الحضور، كما يوفر لوحة تحكم متكاملة للمسؤول لإدارة جميع بيانات النظام.

---

# فكرة المشروع

يهدف المشروع إلى تحويل إدارة المؤسسة التعليمية من النظام الورقي إلى نظام إلكتروني يسهل إدارة الدورات التدريبية والمستخدمين وعمليات التسجيل والحضور.

يوفر النظام واجهتين رئيسيتين:

### المسؤول (Admin)

- إدارة المستخدمين.
- إدارة التصنيفات.
- إدارة الدورات.
- إدارة تسجيلات الطلاب.
- إدارة الحضور.
- لوحة تحكم تعرض بيانات النظام.

### المستخدم

- إنشاء حساب جديد.
- تسجيل الدخول والخروج.
- استعراض الدورات.
- التسجيل في الدورات.
- متابعة الدورات المسجل بها.
- متابعة سجل الحضور.

# طريقة تشغيل المشروع

## 1- تحميل المشروع

قم بنسخ المشروع إلى المجلد:

```
xampp/htdocs/
```

---

## 2- إنشاء قاعدة البيانات

افتح phpMyAdmin ثم أنشئ قاعدة بيانات باسم:

```
alqaser_db
```

---

## 3- استيراد قاعدة البيانات

استورد الملف:

```
database.sql
```

---

## 4- تشغيل خدمات XAMPP

قم بتشغيل:

- Apache
- MySQL

---

## 5- فتح المشروع

افتح الرابط التالي في المتصفح:

```
http://localhost/alqaser_v2/
```
